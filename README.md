# Meridian

Meridian is a focused task tracker built for my final project in IT 718. This project is a sandbox for testing out a priority task board, Cognito-based sign-in, and a simple AWS-backed workflow in a browser app.

> [!NOTE]
> This repository is more of a working project than a polished product. The goal was to explore the technical pieces of a cloud solution rather than a fully fleshed out product.

## What It Is

Meridian gives users a place to sort work by priority, track their progress through the day, and keep a clear view of what still needs attention.

## Project Details

- A clean task board layout built with HTML, CSS, and JavaScript
- Cognito sign-in for a browser-only app
- A lightweight AWS-hosted frontend and API flow
- Simple and responsive interface that stays easy to use
